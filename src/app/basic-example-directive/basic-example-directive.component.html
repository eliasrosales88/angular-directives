Creacion de una directiva basica

Esta directiva tiene el unico objetivo de resaltar parrafos en verde:

La etiqueta con la directiva es:

< p appBasic>Texto... < /p>

El resultado de la etiqueta es:

Este parrafo ha sido resaltado en verde ya que tiene una directiva que accede a la propiedad del elemento donde que se le coloca la directiva

El codigo de la directiva es sencillo:

{{"import { Directive, ElementRef, OnInit } from '@angular/core';"}}
{{"@Directive({
    selector: '[appBasic]'
  })"}}
{{"export class BasicDirective implements OnInit {

      constructor( private elementRef:ElementRef) { }
    
      ngOnInit(){
        this.elementRef.nativeElement.style.backgroundColor = 'lightgreen';
      }
    }"}}
