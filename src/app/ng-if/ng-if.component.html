Las directivas de estructura crean o eliminan elementos del DOM. Un elemento que use ngIF no existira en el DOM a menos que su condicion se cumpla. No estara oculta, simplemento no existira.

Para realizar esta funcion se usa la directiva ngIf en la plantilla html de la siguiente manera:

< p *ngIf="paragraphCreated; else noParagraph" >El contenido del parrafo es: {{ "paragraphContent" }}< /p >

Donde "paragraphCreated" es una propiedad de la clase NgIfComponent:

{{'export class NgIfComponent implements OnInit {'}}
{{'paragraphCreated = false;'}}
{{'}'}}
